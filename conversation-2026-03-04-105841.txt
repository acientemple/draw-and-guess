> 如何把这个游戏部署到github中，让在线就可以玩。 

● 这个项目是纯静态网页，很适合用 GitHub Pages 托管。以下是部署步骤：

  步骤 1: 创建 GitHub 仓库

  1. 访问 https://github.com/new
  2. 仓库名称填 draw-and-guess（或你喜欢的名字）
  3. 选择 Public
  4. 点击 "Create repository"

  步骤 2: 上传文件

  在仓库页面点击 "uploading an existing file"，把以下文件拖进去：
  - index.html
  - css/style.css
  - js/script.js

  步骤 3: 启用 GitHub Pages

  1. 进入仓库 → Settings → Pages
  2. Source 选择 Deploy from a branch
  3. Branch 选择 main（或 master）
  4. 点击 Save

  步骤 4: 访问

  等待 1-2 分钟后，访问 https://你的用户名.github.io/draw-and-guess/
