# 你画我猜 - 网页版游戏

这是一个基于HTML5 Canvas和JavaScript的在线"你画我猜"游戏。

## 功能特点

- 实时绘画功能
- 响应式设计，适配不同屏幕尺寸
- 多人游戏界面（模拟）
- 计分系统
- 聊天功能
- 撤销/重做功能
- 多种颜色和笔刷大小选择

## 如何运行

1. 将整个项目文件夹下载到本地
2. 在浏览器中打开 `index.html` 文件即可开始游戏

## 技术栈

- HTML5 Canvas - 用于绘图功能
- CSS3 - 样式和响应式布局
- JavaScript (ES6) - 游戏逻辑和交互

## 文件结构

```
draw-and-guess/

├── css/
```

## 游戏操作说明

### 绘画功能
- 在中央画布区域点击并拖拽进行绘画
- 使用左侧颜色按钮选择画笔颜色
- 使用滑块调整笔刷大小
- "清空画布"按钮可清除所有内容
- "撤销"按钮可撤销上一步操作

### 聊天功能
- 在底部输入框输入文字并点击"发送"或按回车键发送消息
- 所有玩家的消息会显示在右侧聊天区域

### 游戏规则（模拟）
- 系统随机选择一名玩家作为画家
- 画家需要根据提示词进行绘画
- 其他玩家通过聊天窗口猜测图画内容
- 猜对的玩家和画家都会获得分数
